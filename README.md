# 🚀 소개

- 인덕대학교 IT 웹개발 동아리 모던 애자일 10기 메인 프로젝트 '폴리카소(Pollycasso)' 프론트엔드 레포지토리입니다.

# 🛠️ 사용 라이브러리 & 스택

- [React](https://github.com/facebook/react): UI 컴포넌트 기반 프레임워크, SPA 아키텍처
- [TypeScript](https://github.com/microsoft/TypeScript): 정적 타입 시스템으로 코드 안정성과 유지보수성 강화
- [Axios](https://github.com/axios/axios): HTTP 클라이언트, 인증 토큰 처리 및 API 통신에 사용
- [react-router](https://github.com/remix-run/react-router): 클라이언트 사이드 라우팅, 웹툰 상세/에피소드/마이페이지 네비게이션 구현
- [MSW](https://github.com/mswjs/msw): Mock Service Worker, 개발 단계에서 API 응답 시뮬레이션
- [Modern-kit](https://github.com/modern-agile-team/modern-kit): 클라이언트 개발에 유용한 리액트 컴포넌트, 커스텀 훅 및 유틸리티 함수, 타입을 제공하는 라이브러리
- [dayjs](https://github.com/iamkun/dayjs): 날짜·시간 처리 라이브러리, 업로드일/댓글 시간 포맷에 활용
- [Prettier](https://github.com/prettier/prettier): 코드 포맷터, 일관된 코드 스타일 유지
- [react-hook-form](https://github.com/react-hook-form/react-hook-form): 폼 상태 관리 라이브러리, 회원가입/로그인 같은 입력 검증을 간단하고 효율적으로 구현
- [TanStack Query](https://github.com/TanStack/query): 서버 상태 관리 라이브러리, API 요청 캐싱 및 동기화로 데이터 fetching 최적화
- [Zustand](https://github.com/pmndrs/zustand): 가벼운 전역 상태 관리 라이브러리, 게임 진행 상황이나 사용자 세션 상태 공유에 활용
- [Socket.IO](https://github.com/socketio/socket.io): 실시간 양방향 통신 라이브러리, 그림 대결 게임의 실시간 이벤트 처리에 사용
- [react-konva](https://github.com/konvajs/react-konva): Canvas 기반 그래픽 라이브러리, 그림판 UI 및 드로잉 기능 구현에 최적
- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss): 유틸리티 퍼스트 CSS 프레임워크, 빠른 스타일링과 반응형 UI 구현
- [Framer Motion](https://github.com/motiondivision/motion): 애니메이션 라이브러리, 페이지 전환과 UI 인터랙션을 자연스럽게 연출

# 📝 커밋 메시지 컨벤션

- `feat` : 새로운 기능 추가
- `fix` : 버그 수정
- `docs` : 문서 수정
- `style` : 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
- `refactor` : 코드 리펙토링
- `test` : 테스트 코드, 리펙토링 테스트 코드 추가
- `chore` : 빌드 업무 수정, 패키지 매니저 수정
- `conflict`: 충돌 해결
